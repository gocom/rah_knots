h1. rah_knots

*This functionality is now part of current Textpattern CMS versions. This plugin is not needed and is deprecated.*

"Download":https://github.com/gocom/rah_knots/releases | "Packagist":https://packagist.org/packages/rah/rah_knots | "Issues":https://github.com/gocom/rah_knots/issues

Save "Textpattern CMS'":https://textpattern.com editors with two familiar keyboard shortcuts, Command+S and CTRL+S.

h2. Installing

Using "Composer":https://getcomposer.org:

bc. $ composer.phar require rah/rah_knots

Or "download":https://github.com/gocom/rah_knots/releases an installer package.
